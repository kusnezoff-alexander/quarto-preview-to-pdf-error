# scientific_writing_skeleton

Usage: `quarto preview` (for previewing paper in browser)
- `quarto render --to pdf`

Structure:
- `/content`=actual content of paper/thesis
- `/img`=images used inside paper/thesis
- `references.bib`=bib-file of references which are cited in this work
- `ieee.cls`=citation-style to use
